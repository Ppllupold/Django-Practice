# 📝 Django Practice App

A simple Django-based task manager app for practicing full-stack development.

## 🚀 Features

- Create, update, delete tasks ✅
- Assign multiple tags to tasks 🏷️
- Mark tasks as complete or not done ✅/❌
- Tag management page with CRUD actions
- Responsive layout using Bootstrap 4

## 🧱 Tech Stack

- Django 5.x
- Bootstrap 4
- Crispy Forms (`crispy-bootstrap4`)
- Python 3.11+
